# Smart Expense Tracker

A simple web application built using **Python and Django** to track daily expenses.

## Features

* Add daily expenses with description, amount, category, and date
* Store expense data in a database
* Display expenses in a table
* Simple and easy-to-use interface

## Technologies Used

* Python
* Django
* HTML
* SQLite Database

## Project Structure

```
SmartExpenseTracker
│
├── expense_tracker
├── tracker
│   └── templates
│       └── tracker
│           └── home.html
├── manage.py
└── db.sqlite3
```

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/rishanaaslam945-droid/Smart-expenses-tracker.git
```

2. Go to the project folder

```
cd Smart-expenses-tracker
```

3. Install Django

```
pip install django
```

4. Run the server

```
python manage.py runserver
```

5. Open in browser

```
http://127.0.0.1:8000/
```

## Project Description

The **Smart Expense Tracker** helps users manage and record their daily expenses.
Users can enter the description, amount, category, and date of an expense.
The application stores the data in a database and displays it in an organized table.
